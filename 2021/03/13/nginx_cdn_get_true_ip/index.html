nginx如何获取用户的真实IP地

使用CDN后，用户的请求是经过CDN处理后得到的，因此需要获取用户的真实IP地址，需要用到realip模块。可以在安装的时候添加–with-http_realip_module选项，这样就能添加该模块；也可以通过nginx -V查看是否已经添加了该模块。
nginx http块配置：

配置解析

  • set_real_ip_from：设置可信任的IP地址
  • real_ip_header：从指定的请求头中获取客户端IP地址，此请求头一般用$X-Forwarded-For
  • real_ip_recursive： 如果设置为on，则表示启用递归查询，realip_module在获取客户端地址时，会在real_ip_header指定的请求头中逐个匹配IP，不在可信任的IP地址则作为客户端地址。

IP地址伪造

有些恶意攻击会伪造这个请求头进行访问，通过干扰服务器的判断来隐藏真实IP。可以通过以下的手段解决：

  • 和CDN约定一个新的请求头
  • 在CDN代理客户端请求时，要求CDN传递用户真实IP，并且建立连接时，通过CDN清除伪造的请求头。
